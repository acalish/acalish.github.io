GameTracker

GameTracker is a full stack app to track board games and comments. One of my favorite things to do is play board games, but there are so many, and they often have really obscure / similar / confusing names. I wanted to make an app where people can keep track of games, and write comments about them. In future versions, I would want there to be more information about each game, and for users to also have the ability to create a wishlist of games. I built an API that is hosted on heroku, and a front end where registered users can create games. See my live client site in action. Want to see my code?
